Currently support sites

You can view a list of the currently supported sites

Run the map data compactor

Each sites has its own map file. This compacting tool builds them into a single compress file for use by the module.

Creating the sprites/css from the icons

Install a copy of glue onto your Mac and in the project folder run $ glue --html --optipng --watch icons sprites.
